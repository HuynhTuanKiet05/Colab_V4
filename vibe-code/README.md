# Vibe Code Workspace

Cấu trúc thư mục này được tạo để làm việc theo kiểu "vibe code" với các file agent rõ ràng, dễ mở rộng và dễ giao việc cho AI.

## Cấu trúc

- `agents/` - các file mô tả vai trò của từng agent
- `prompts/` - các prompt dùng lại cho từng tác vụ
- `notes/` - ghi chú dự án, TODO, checklist

## Mục đích

- Tách vai trò rõ ràng: planner, coder, reviewer
- Dễ thêm workflow cho các dự án khác nhau
- Giữ prompt ngắn, nhất quán và tái sử dụng được
