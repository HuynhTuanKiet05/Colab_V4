## 3) Quyết định về cách kết nối model với web

### Cách đã chọn
Mình chọn **cách số 2**:
- chạy Python API riêng
- PHP gọi sang Python qua `http://127.0.0.1:8000`

### Luồng dự đoán
1. Người dùng nhập thuốc hoặc bệnh trên web PHP
2. PHP gửi request sang Python API
3. Python API trả về:
   - danh sách top-k
   - score
   - matched input
   - graph nodes/links
4. PHP lưu lịch sử vào MySQL
5. Frontend hiển thị bảng kết quả và graph 3D

### Ý nghĩa
- phần AI có thể thay đổi sau mà không ảnh hưởng giao diện web
- web vẫn chạy được trên XAMPP theo kiểu rõ ràng, dễ bảo trì
